The Inventory page in the JKUAT Inventory Management System lets you browse all available stock items at JKUAT. You can search by item name, filter by category or department, and view details like current quantity and unit of measure before submitting a request.

## Accessing the Inventory

From your dashboard, click **Inventory** in the navigation menu. You will see a paginated table (or card grid, depending on screen size) of all available inventory items.

## Inventory Table Columns

| Column | Description |
|---|---|
| Item Name | The name of the inventory item |
| Category | The category it belongs to (e.g. Stationery) |
| Department | The department that holds the item |
| Available Quantity | Current stock available for requests |
| Unit | Unit of measure (e.g. Pieces, Boxes) |
| Status | In Stock, Low Stock, or Out of Stock |
| Action | Request button (only active for In Stock items) |

<Note>
  You can view all items regardless of stock status, but you can only submit a request for items that are **In Stock**. Out of Stock items are shown for reference only.
</Note>

## Searching for an Item

Use the search bar at the top of the inventory table:
1. Type the item name or a keyword (e.g. "paper", "printer").
2. Results filter in real time as you type.
3. Clear the search field to return to the full list.

## Filtering Inventory

Use the dropdown filters to narrow the list:

| Filter | Options |
|---|---|
| Category | All Categories or a specific category |
| Department | All Departments or a specific department |
| Status | All, In Stock, Low Stock |

Filters can be combined. For example, filter by Category = "Electronics & ICT" and Status = "In Stock" to see only available electronic items.

## Viewing Item Details

To see full details about an item, click its name in the table. The item detail view shows:
- Full description
- Current quantity and minimum stock level
- Department and category
- Date added to the system

## Requesting an Item

From the inventory table or item detail view, click the **Request** button on any In Stock item to open the request form. See [Submitting a Request](/user/requests) for full instructions.

<Tip>
  If you cannot find an item, it may be listed under a different name or category. Try a shorter search keyword or browse by category using the filter.
</Tip>
