The JKUAT Inventory Management System is a modern, web-based platform built exclusively for Jomo Kenyatta University of Agriculture and Technology. It gives administrators full control over university assets, stock levels, and user access, while providing staff and departmental users a clean interface to view inventory and submit stock requests — all in one unified system.

## Getting Started

Follow these steps to get up and running with the JKUAT Inventory Management System.

<Steps>
  <Step title="Check System Requirements">
    Confirm your server environment meets the minimum requirements — PHP 7.4+, MySQL 5.7+, and a web server such as Apache or XAMPP. See [System Requirements](/system-requirements).
  </Step>
  <Step title="Install the System">
    Download the project files, configure your database connection, import the SQL schema, and launch the application. See [Installation](/installation).
  </Step>
  <Step title="Log In as Administrator">
    Use the default admin credentials to access the admin dashboard. From there you can set up departments, categories, and create user accounts for your staff.
  </Step>
  <Step title="Onboard Your Users">
    Create login credentials for each user through the admin panel. Users receive their credentials from the administrator — they cannot self-register.
  </Step>
</Steps>
